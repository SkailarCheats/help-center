---
title: "System Requirements"
---


Before installing Skailar cheats, ensure your system meets the following requirements to ensure optimal performance and compatibility.

### Minimum System Requirements

- **Processor:** Intel or AMD processor
- **Operating System:** Windows 10/11 (Version 22H2 or later)
- **Windows Build:** Standard Windows 10/11 installations (not compatible with custom builds like Ghost Spectre or other custom Windows versions)
- **Additional Requirements:** Secure Boot must be disabled in BIOS for proper functionality.

### Recommended System Specifications

- **Processor:** Intel Core i5 / AMD Ryzen 5 or higher
- **RAM:** 8 GB or more
- **Graphics:** NVIDIA GTX 1060 / AMD Radeon RX 580 or equivalent
- **Storage:** SSD recommended for faster loading times

<Warning>
NOTE: Skailar cheats are designed to work on standard Windows 10/11 installations with the specified Windows build (22H2 or later). Custom Windows builds such as Ghost Spectre or other modified versions may not be supported.
</Warning>
